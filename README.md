# Pizza Sales Dashboard (Power BI)

This project analyzes one year of pizza sales data for a fictional pizza chain and turns it into an interactive Power BI dashboard. It helps stakeholders understand revenue drivers, busy periods, and top/bottom performing pizzas.

## Files

- `Pizza_Sales.pbix` – main Power BI report
- `Pizza_Dashboard_1.png` – overview page screenshot
- `Pizza_Dashboard_2.png` – trends page screenshot

## Key Insights

- Total revenue, total orders, total pizzas sold, and average order value
- Top and bottom pizzas by revenue, quantity, and number of orders
- Daily and monthly order trends, plus sales share by category and size

## Tools & Skills

- Power BI Desktop, DAX
- Data modeling (star schema)
- Basic data cleaning in SQL/Excel

## How to Use

1. Download `Pizza_Sales.pbix`.
2. Open it in **Power BI Desktop**.
3. Use the slicers (date range, pizza category) to explore the dashboards.
